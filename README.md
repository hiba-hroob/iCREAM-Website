iCREAM is a modern and responsive ice cream website developed using HTML, CSS, and JavaScript.
It includes interactive features such as scroll animations, animated statistics counters,
and a flavor suggestion system to enhance the user experience.
The project demonstrates clean UI design, responsive layout techniques, and dynamic front-end functionality.
